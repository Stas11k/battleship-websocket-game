# Морський бій на WebDataRocks

Проєкт реалізований на HTML, CSS і JavaScript.

- `client/` — клієнтська частина: HTML, CSS, JavaScript, WebDataRocks.
- `server/` — серверна частина на JavaScript, яка запускається через Node.js.

Node.js використовується тільки для запуску JavaScript-сервера з WebSocket. Сервер синхронізує двох гравців, приймає розміщення кораблів, ходи та надсилає результат обом клієнтам.

## Що реалізовано

- дві дошки через WebDataRocks;
- підключення двох гравців через WebSocket;
- фаза розміщення кораблів;
- кожен гравець сам розміщує 10 корабельних клітинок на своєму полі;
- постріли по полю суперника;
- перевірка попадання на сервері;
- синхронізація двох дошок;
- завершення гри після знищення всіх кораблів суперника.

## Запуск сервера

```bash
cd server
npm install
npm start
```

Сервер запускається на:

```text
ws://localhost:3001
```

## Запуск клієнта

Відкрийте `client/index.html` через Live Server у двох вкладках браузера.

- Перша вкладка — player1.
- Друга вкладка — player2.

Після підключення двох гравців починається фаза розміщення кораблів. Кожен гравець натискає по своєму лівому полю і розміщує 10 корабельних клітинок. Після цього гра автоматично переходить до фази бою.
